A professional, research-focused digital resume showcasing expertise in **econometrics**, **causal inference**, **machine learning for policy evaluation**, and **business analytics**. Designed for PhD applications and research positions.

## 🎯 Design Philosophy

This resume is crafted specifically for a **Business Analyst / Industrial Economist**:
- **Clean, data-driven layout** with metrics and impact visualization
- **Professional navy & teal color palette** reflecting trust and analytics
- **Skill proficiency bars** showing technical competency levels
- **Research impact stats** highlighting key achievements
- **Publication highlights** with visual icons
- **PDF download option** for recruiters

## 🚀 Features

- **Hero Section**: Professional introduction with skill badges
- **Contact Bar**: All social/professional links in one place
- **Two-Column Layout**: Optimal information hierarchy
- **Experience Timeline**: Clear role progression with achievements
- **Skills Visualization**: Progress bars for programming languages
- **Research Metrics**: Key impact numbers highlighted
- **Publication List**: Academic and professional writing
- **Reference Section**: Academic advisors contact info
- **Responsive Design**: Works on all devices
- **Print/PDF Friendly**: Optimized for downloading

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Font Awesome Icons
- Google Fonts (Inter + Playfair Display)
